### RUN THE PROJECT

```shell
pip install pipenv
```
**RUN FROM THE PROJECT DIRECTORY WITH THE FOLLOWING FILES :**
1. Pipfile
2. Pipfile.lock

```shell
pipenv install
```
**ACTIVATE THE PROJECT**
```shell
pipenv shell
```
